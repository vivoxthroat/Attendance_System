# Smart Attendance System using Face Recognition

## Overview

This project is an AI-powered attendance management system that automatically detects and identifies students using facial recognition.

The system records attendance with timestamps and exports the final report into CSV format.

## Features

- Face Detection using OpenCV
- Face Recognition using MobileNetV2
- Attendance Logging
- CSV Report Generation
- Time-based Attendance Window
- Present and Absent Student Tracking
- Custom Dataset Training

## Technologies Used

- Python
- OpenCV
- TensorFlow
- NumPy
- Pandas
- Scikit-Learn

## Workflow

1. Capture student images
2. Train face recognition model
3. Detect faces from webcam
4. Identify students
5. Mark attendance
6. Export CSV report

## Dataset

Custom dataset collected using webcam.
